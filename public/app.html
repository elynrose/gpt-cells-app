üéØ Getting Started

GPT Cells is an AI-powered spreadsheet that generates text, images, and audio content.

  • Click any cell to start typing your prompt
  • Select a model from the dropdown (GPT, DALL-E, TTS)
  • Run the cell to generate content
  • Click the expand button (‚õ∂) to see full generation history
  • Right-click row/column headers to insert or delete rows/columns

‚å®Ô∏è Keyboard Shortcuts

  • Ctrl+Enter - Run current cell
  • Ctrl+Shift+R - Run all cells
  • Ctrl+Shift+C - Clear all cells
  • Ctrl+Z - Undo
  • Ctrl+Y - Redo
  • Escape - Close modals/outputs

ü§ñ AI Models

  • GPT Models: Generate text content

üìä Features

  • Cell Dependencies: Use {{A1}} to reference other cells
  • Cross-Sheet References: Use {{Sheet2!A1}} to reference cells from other sheets
  • Automatic Execution: Dependent cells run automatically when referenced cells are updated
  • Generation History: View all previous generations in the modal
  • Multiple Sheets: Create and manage multiple spreadsheets
  • Persistent Storage: All data is automatically saved

üìä Row & Column Management

  • Add rows/columns: Use + Row/+ Column buttons or right-click headers
  • Delete rows/columns: Use - Row/- Column buttons or right-click headers
  • ‚ö†Ô∏è Warning: Deleting rows/columns with data or references will show a warning
  • Minimum size: At least 1 row and 1 column must remain
  • Data preservation: Deletion shifts remaining data up/left

üí° Tips

  • Use specific prompts for better results
  • Experiment with different temperature values (0.1-2.0)
  • Combine text and image generation in the same workflow
  • Use cell references to build complex AI workflows
  • Create cross-sheet workflows: {{Sheet1!A1}} + {{Sheet2!B1}}
  • Watch the execution order when dependencies are updated
